## 🌱 About ZU X ORI

**ZU X ORI** is a mythic narrative told across lifetimes — a reincarnation-era sequel to *Romeo and Juliet*, unfolding through memory, mystery, and the re-writing of fate.

This repository is part of the wider [ZU X ORI universe](https://github.com/zuxori), which includes research, story episodes, visual assets, and conceptual threads meant to be explored and expanded.
